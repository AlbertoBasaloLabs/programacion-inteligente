# Cuerstionario bloque 1. Módulos 1 a 3.

## 1. ¿En qué consiste principalmente el concepto de "Vibe Coding"?
- a) Escribir código optimizado manualmente
- b) Usar solo editores de texto plano
- c) Desarrollar siguiendo el flujo sin preocuparse por el código
- d) Programar sin usar inteligencia artificial

> Ok: c

## 2. En el ecosistema de IA, ¿qué función principal cumple un "Modelo"?
- a) Ejecutar tareas específicas como un agente
- b) Interactuar con el código fuente directamente
- c) Proveer herramientas de edición de texto
- d) Procesar lenguaje natural y generar texto

> Ok: d

## 3. ¿Cuál de las siguientes opciones se considera una "primitiva" en la programación con IA?
- a) MCPs (Model Context Protocol)
- b) Compiladores
- c) Sistemas operativos
- d) Navegadores web

> Ok: a

## 4. En el flujo de Vibe Coding, ¿quién es el responsable final de la validación?
- a) El modelo de IA
- b) El humano
- c) El editor de código
- d) El sistema de integración continua

> Ok: b

## 5. ¿Por qué es crucial proporcionar instrucciones claras a los LLM?
- a) Porque los LLM no pueden funcionar sin ellas
- b) Para ahorrar costes de procesamiento
- c) Porque la IA inventará o completará por defecto lo que falte
- d) Porque así lo exige el compilador

> Ok: c

## 6. ¿Qué nivel de jerarquía de instrucciones se aplica específicamente a ficheros o carpetas seleccionados?
- a) Organización
- b) Usuario
- c) Proyecto
- d) Patrón

> Ok: d

## 7. ¿Cuál es la premisa principal de "Prompt as Code"?
- a) Que las peticiones sean reutilizables y mantenibles como el código
- b) Que los prompts se escriban en binario
- c) Que los prompts sean de un solo uso
- d) Que no se usen comentarios en los prompts

> Ok: a

## 8. ¿Qué permiten los MCPs a los agentes aumentados?
- a) Mejorar la velocidad de procesamiento
- b) Extender capacidades e interactuar con el mundo
- c) Reducir el coste de la API
- d) Generar imágenes exclusivamente

> Ok: b

## 9. ¿Cómo se define un "Agente" en el contexto de la programación inteligente?
- a) Un editor de texto avanzado
- b) Una base de datos vectorial
- c) Un programa que usa modelos para cumplir tareas específicas
- d) Un lenguaje de programación nuevo

> Ok: c

## 10. ¿Cuál es la solución propuesta para manejar tareas complejas y lentas en la programación inteligente?
- a) Trabajar en paralelo y en la nube
- b) Simplificar las tareas al máximo
- c) Usar ordenadores más potentes localmente
- d) Evitar tareas complejas

> Ok: a
