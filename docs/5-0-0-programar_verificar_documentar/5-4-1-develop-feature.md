# Develop Workflow

- Develop the next feature from PRD.

## Workflow

- **Product-Owner** : Writes the specification for the feature 
- **Engineer** : Prepares the implementation plan and coding repository
- **Coder** : Writes code to implement the plan with unit test and best skilled practices. 
- **Tester** : Writes and runs tests to verify a specification implementation
- **Cleaner** : Simplifies and refines implementation for maintainability.
- **DevOps** : Writes documentation and release the implementation.
