# Cuestionario bloque 2. Módulos 4 a 6.

## 1. ¿Cuál es el punto de partida del flujo de trabajo Spec-Driven Development (SDD)?
- a) El código fuente existente
- b) La interfaz de usuario
- c) Las especificaciones
- d) La base de datos

> Ok: c

## 2. ¿Qué agente es el responsable principal de generar el Documento de Requerimientos del Producto (PRD)?
- a) Coder Agent
- b) Tester Agent
- c) Architect Agent
- d) Analyst Agent

> Ok: d

## 3. ¿Cuál es el propósito principal de la "planificación de implementación" en el módulo 4?
- a) Dividir las especificaciones en tareas manejables
- b) Escribir el código final inmediatamente
- c) Desplegar la aplicación en producción
- d) Diseñar el logotipo de marketing

> Ok: a

## 4. Según el curso, ¿qué parte del proceso de desarrollo se considera "la parte fácil" al usar IA?
- a) Verificar que funciona
- b) Generar el código
- c) Documentar los cambios
- d) Recopilar los requerimientos

> Ok: b

## 5. ¿Cuál es la función principal del "Tester Agent"?
- a) Escribir y ejecutar pruebas para el código implementado
- b) Escribir las especificaciones funcionales
- c) Generar el código inicial de la aplicación
- d) Gestionar la infraestructura en la nube

> Ok: a

## 6. ¿Por qué se considera crítica la documentación de los cambios (Changelog, actualización de artefactos)?
- a) Porque nadie la lee realmente
- b) Porque es un requisito legal
- c) Para aumentar el tamaño de los archivos del repositorio
- d) Para mantener la sostenibilidad y mantenibilidad del producto

> Ok: d

## 7. ¿Cuál es la estrategia recomendada para manejar la deuda técnica con ayuda de la IA?
- a) Ignorarla hasta que el sistema falle
- b) Refactoring inteligente continuo
- c) Reescribir todo el proyecto desde cero cada mes
- d) Solo arreglar los bugs reportados por usuarios

> Ok: b

## 8. ¿Qué permite la "Orquestación Multi-Agente" en el contexto de la productividad?
- a) Ejecutar solo una tarea a la vez para evitar errores
- b) Evitar el uso de agentes de IA
- c) Ejecución en paralelo de tareas como codificar, testear y documentar
- d) Reducir la necesidad de computación en la nube

> Ok: c

## 9. ¿A qué se aplica el "Ciclo Virtuoso" de mejora continua en el módulo 6?
- a) A las primitivas (prompts, skills, specs) basándose en el feedback
- b) Al hardware utilizado para el desarrollo
- c) Únicamente al código final compilado
- d) Al mobiliario de la oficina

> Ok: a

## 10. ¿Qué rol juega el "Meta Agent" en el ecosistema de programación inteligente?
- a) Desplegar el código a producción
- b) Gestionar el presupuesto del proyecto
- c) Escribir el PRD inicial
- d) Mejorar instrucciones y primitivas usando la propia IA
