## Sample project : Astro-Bookings
API mínima en TypeScript y Express .

### Flujo de desarrollo
- Instalar dependencias: `npm install`
- (Una vez) Instalar navegadores para e2e: `npx playwright install`
- Servir en desarrollo (usa ts-node): `npm run dev`
- Build a JavaScript: `npm run build`
- Ejecutar build (prestart compila): `npm start`
- Tests end-to-end (Playwright): `npm test`
- Endpoint de salud: `GET http://localhost:3000/health` devuelve `{ status: "ok", timestamp: "..." }`
